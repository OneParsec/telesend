# telesend
Telegram Spamer. Use lib telethon. Write by Python.

First you need to go to my.telegram.org/apps and sign in.

You need to create new application.

Copy api_id and api_hash to api_id.txt and api_hash.txt

Download repo.
```$ git clone https://github.com/OneParsec/telesend```

Paste api_id to api_id.txt and api_hash to api_hash.txt

Install requirements
```$ pip install -r requirements.txt```

And start a program
```python telesend.py```
